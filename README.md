How to use :

	1- Compile :

		 ** IT SHOULD LOOK PRETTY **

		$> make

		OR

		** IF THE SPECIAL SYMBOL DOES NOT WORK

		$> make ascii

	2- Run :

		usage:
		$> 	game

		OR

		$> game level_path

		Note that your map must width and height must be greater than 30
		It is recommend to use the main.lvl map as Template


		example:
		$> game ./level/main.lvl
